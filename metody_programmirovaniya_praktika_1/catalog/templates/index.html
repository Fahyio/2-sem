О проекте "Каталог курсов"

Этот сайт был разработан в рамках Практической работы №1 по Django.

📋 Что было сделано:

  • ✅ Установлен Django и создано виртуальное окружение
  • ✅ Создан Django-проект с приложением "catalog"
  • ✅ Реализована архитектура MVT (Model-View-Template)
  • ✅ Настроена маршрутизация (URLs) с view-параметрами
  • ✅ Созданы все необходимые страницы без использования БД
  • ✅ Данные хранятся в файле data.py в виде констант

🌐 Доступные страницы:

  • Главная - описание проекта (эта страница)
  • Курсы - список всех доступных курсов
  • Страница курса - детальная информация о конкретном курсе
  • Авторы - список авторов курсов
  • Страница автора - информация об авторе и его курсах
  • О проекте - общая информация о сайте
  • 404 страница - страница "Не найдено" для несуществующих URL

🛠 Технологии:

  • Python 3.12
  • Django 6.0
  • HTML/CSS
  • Django Template Language

Проект создан для изучения основ веб-разработки на Django.
