Учебный проект для изучения Django и веб-разработки

🎯 Цель проекта

Этот сайт был разработан в рамках Практической работы №1 по Django для изучения основных концепций веб-разработки:

  • Архитектура MVT (Model-View-Template) в Django
  • Маршрутизация (URL dispatcher) с параметрами
  • Работа с шаблонами Django Template Language
  • Создание многостраничных веб-приложений
